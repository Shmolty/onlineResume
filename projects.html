Network Installation of Windows 11

I found this to be one of the coolest projects that I have done. I had to configure the BIOS settings to boot using PXE, and then connect the computer to the network so that it can reach the server providing PXE boot. From there it was following typical Windows installation steps.

I also set up installations via ISO file on a USB drive for both Windows and Linux
